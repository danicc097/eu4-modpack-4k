[h1][b]Welcome to Advisor Types Expanded[/b][/h1]
This mod adds new advisor types and events related to them to add more variety and depth to the advisor system.

[h1]Currently Added[/h1]
Currently [i]Advisor Types Expanded[/i] adds [b]74[/b] new advisor types.
The full list with the modifiers can be found [url=https://steamcommunity.com/workshop/filedetails/discussion/2737385499/4944391335293065029/]here[/url].
